# Обработка на грешки
- **Невалиден ЕГН:** базова проверка за 10 цифри; при грешка — inline съобщение на формата.
- **Невалиден домейн за специалист:** ако `@ucb.bg` email е използван, но няма специалист по категорията на жалбата → fallback на централна опашка с `assignedTo: null` и `status: registered`.
- **Gemini грешка / timeout:** специалистът вижда бутон „Generate draft" с възможност за ручно писане на отговор; жалбата не се блокира.
- **SLA наближаване:** Cloud Function scheduler (cron) маркира жалби с `slaDueAt - now < 1 работен ден` с `slaWarning: true` — UI бейдж в опашката на специалист и в supervisor dashboard-а.
- **Опит за достъп до защитен път:** PrivateRoute пренасочва към подходящата начална страница по роля.
- **Опит за смяна на status извън разрешен преход:** Firestore Rules отказват write; UI показва toast „Не е разрешен преход".
- **Монетарно действие без одобрение:** rule блокира write; UI показва „Необходимо одобрение от ръководител".
- **Оспорване без причина:** форма изисква минимум 20 символа в `disputeReason`.
